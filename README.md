# Arduino Alarm Clock 


#### Designed, assembled and programmed an alarm clock intended to wake users with sound and lights

### Materials 
* Arduino UNO
* RTC Module
* Standard 16x2 LCD
* Active buzzer
* IR Receiver
* remote 
* 12V White LED
* 12V battery
* 100K resistor
* Potentiometer
